# Symbol 
the symbol of the Dirac Delta Function is the $\delta(t)$

1) $\delta(x)=0, x\not=0$
2) $\int_{-\infty}^{\infty}\delta(x)dx=1$ 
# Properties 

## Sifting 
For every $\phi(t)$ which is continuous at $t = 0$
$\int_{-\infty}^{\infty}\phi(t)\delta(t)dt=\phi(0)$
